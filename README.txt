Ручная компиляция
Положить файлы main11.cpp, course_lib.cpp, course_lib.h рядом
Скомпилировать динамическую библиотеку:
g++ -shared -fPIC -o course_lib.so course_lib.cpp
Скомпилировать программу:
g++ -Wl,-R . -o course main11.cpp course_lib.so

Запуск программы
./course

Программа выполняет сортировку целых чисел из файла (по умолчанию - input.txt)
Параметр -q заставляет использовать быструю сортировку вместо пузырьковой.
Параметр -in=somefile.txt - использовать последовательность из somefile.txt, а не из input.txt
Параметр -out=somefile.txt - сохранять результаты в somefile.txt, а не в output.txt

Имеется генератор тестов (входных последовательностей размера 2^N).
Для его запуска есть параметр -test=N, где N = [1; 31]
Создаётся три файла test-N-: случайная последовательность (rnd), отсортированная по возрастанию (asc), по убыванию (desc)
